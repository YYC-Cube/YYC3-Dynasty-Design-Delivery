# 图例说明

## 结构标记

```
符号     说明
══════   ════════════════════════════════════════
#        一级标题（Part）
##       二级标题（Section / §）
###      三级标题（Sub-section）
├─ └─    树形结构
┌─┐└─┘├─┤  面板/UI 区域
```

```
符号     说明
══════   ════════════════════════════════════════
✅       已完成 / 通过

⏳       待办 / 等待
✕        拒绝 / 驳回
⚡       紧急 / 告警
✓        成功 / 有效
```
