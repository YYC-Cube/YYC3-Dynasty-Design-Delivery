## 文档版本号规范

遵循语义化版本号（SemVer 2.0.0）：

```
v{major}.{minor}.{patch}-{stage}

变更类型:
  - PATCH (+1): 内容修正 / Bug修复
  - MINOR (+1): 新增内容 / 向下兼容变更
  - MAJOR (+1): 结构重组 / 不兼容变更
```
