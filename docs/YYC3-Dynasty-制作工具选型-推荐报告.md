## 二、首推：Kling 3.0（可灵 AI）

### 核心理由

| 本项目需求 | Kling 3.0 能力 | 匹配度 |
|-----------|---------------|:------:|
| 中华古风建筑/服饰/器物 | 中国团队训练，古风场景表现力业界第一 | ⭐⭐⭐⭐⭐ |
| 11 场景多镜头统一叙事 | Multi-Shot Storyboarding 原生支持 | ⭐⭐⭐⭐⭐ |
| 暗金史诗调性 | 4K + 风格迁移 + cinematic mode | ⭐⭐⭐⭐ |
| 中文旁白/字幕 | 中文文字渲染 + 中文语音合成 | ⭐⭐⭐⭐⭐ |
| 成本可控 | 66 每日免费 credits + 合理付费 | ⭐⭐⭐⭐⭐ |

### 关键特性详解

**Multi-Shot Storyboarding（3.0 核心升级）**

支持脚本化多镜头叙事，可直接输入完整分镜脚本，自动生成连贯的多镜头序列。11 场景的分镜提示词可以直接输入，无需逐段拼接。

**MVL（Multi-modal Visual Language）框架**

提示词遵循度大幅提升，能够准确理解"玄墨底色 + 鎏金强调 + 朱砂点睛"的精确色彩描述。

**人像/动态表现**

自然的人物动态输出，适用于马的奔跑、玉玺盖章、书法书写等关键场景。

## 三、推荐工作流：Hybrid 策略

如追求最佳品质，推荐 **Kling 3.0 为主 + 专项工具配合**：

```
第1步：场景生成（Kling 3.0）
├─ 导入全部 11 场景提示词（Multi-Shot Storyboard）
├─ 逐场景微调，确保风格统一
└─ 输出 11 段 1080p/4K 视频素材

第2步：专项优化（选配）
├─ 场景06 玉玺微距  → 可补 Runway Gen-4.5（微距最佳）
└─ 场景02 洛阳航拍  → 可补 Veo 3.1（航拍质感最佳）

第3步：后期合成（Premiere Pro / DaVinci Resolve）
├─ 拼接 11 段素材
├─ 叠加 Color Grade（暗金史诗 LUT）
├─ 嵌入旁白（中文男声，48kHz/24bit）
└─ 添加 BGM（管弦 + 编钟 + 古筝）

第4步：交付输出
└─ 4K 16:9 MP4（H.265）+ 短视频裁切版（9:16）
```

## 四、备选方案

### Wan 2.6（通义万相）

**适合场景：** 如果团队更偏好国产全链路方案

- 阿里巴巴出品，2026.06 已发布 2.6 版本
- **Multi-Shot Storytelling** 原生支持多镜头叙事
- 原生中文语义理解，古风 prompt 无需翻译
- 原生音频生成（语音 + 背景音）
- 与阿里云生态集成（存储/API/算力）

### Sora 2（OpenAI）

**适合场景：** 对叙事连贯性要求极高的戏剧性镜头

- 电影语言理解能力突出，叙事连贯性行业领先
- 原生支持 1080p + 音视频同步
- 中文/古风表现不如国产工具，需配后期
- ChatGPT Plus $20/月 / Pro $200/月

### Runway Gen-4.5

**适合场景：** 微距特写、风格迁移、商业级项目

- 微距镜头、玉玺纹理、粒子细节表现最佳
- 4K 输出 + Motion Brush 精确控制
- 商业生产中可靠性最高
