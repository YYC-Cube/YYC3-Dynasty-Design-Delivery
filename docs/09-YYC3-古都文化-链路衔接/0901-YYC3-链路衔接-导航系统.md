# 全局导航系统

## 一、导航栏设计（所有页面顶部通栏）

```
┌──────────────────────────────────────────────────────────────┐
│ [YYC³ 龙纹Logo]  朝堂  王朝  Skills  旨意    🔍  🔔(3)  👤 │
│                                                              │
│    朝堂 · 紫微正衙              ┌───────────────────────┐    │
│                                │  🔔 通知中心           │    │
│                                │ ───────────────────── │    │
│                                │ 🕊️ 敕令#042 执行完毕  │    │
│                                │ 🔔 敕令#043 门下封驳   │    │
│                                │ ⚡ 令牌不足·户部告警   │    │
│                                │ [+2 条折叠通知]        │    │
│                                └───────────────────────┘    │
└──────────────────────────────────────────────────────────────┘
```

| 属性 | 值 |
|------|-----|
| 高度 | 56px（固定） |
| 背景 | `rgba(10,14,23,0.95)` + `backdrop-filter: blur(12px)` |
| Logo | 龙纹标识 + "YYC³" |
| 导航项 | 朝堂（P2）/ 王朝（P3）/ Skills（P8）/ 旨意（P6） |
| 右侧 | 搜索按钮(⌘K) + 通知铃铛 + 用户头像 |

## 三、侧边快捷入口（P2 朝堂页专属）

| 位置 | 页面右边缘，垂直排列 |
| 宽度 | 48px |
| 行为 | hover 展开文字标签，click 跳转 |

侧边项（从上到下）：
1. 朝堂（P2）— 当前页高亮
2. 旨意板（P6）
3. 双星桥（P8）
4. 万象枢（P7）
5. 返回顶部

## 四、通知中心入口

| 位置 | 导航栏右侧，铃铛图标 |
| 角标 | 未读通知数，最大 "99+" |
| 下拉面板 | 点击展开，显示最近 5 条 + "查看全部" |
| 状态 | 新通知到达时铃铛动画（摇动 3 次，300ms） |
