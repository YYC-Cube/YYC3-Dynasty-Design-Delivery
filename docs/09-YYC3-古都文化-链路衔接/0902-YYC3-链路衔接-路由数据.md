流程序号:    ①         ②         ③         ④         ⑤         ⑥         ⑦

## 三、数据一致性约束

| 约束 | 规则 |
|------|------|
| 阶段不可跳跃 | 1→2→3→4→5→6→7 顺序流转 |
| 不可逆流 | 除"封驳"回到阶段1重拟外 |
| 审议需全签 | 三省签章齐全方可进入派发 |
| 归档只读 | 归档后数据不可修改 |
| 版本号 | 每次更新递增版本号，冲突拒绝写入 |
