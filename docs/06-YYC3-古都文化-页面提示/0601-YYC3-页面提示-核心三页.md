## P2：朝堂大厅 · 紫微中轴线 ⭐ 核心页

**路由**：`/court` | **尺寸**：1440×900
**推荐模型**：Gemini 3.1 Pro（创意复杂布局）

### [Gemini 3.1 Pro] 初始提示词

```
You are a senior UI designer specializing in ancient Chinese imperial architecture
and dark-mode dashboard design. Create the main "Court Hall" page for YYC³ Dynasty.

This page represents the full governance hierarchy of the Three Departments and Six
Ministries system, visualized as the Luoyang Ziwei City central axis.

FRAME: 1440×900, dark theme, no scroll.

LAYOUT (5 layers from top to bottom):

LAYER 1 — NAVBAR (H:56px):
  Dark bg, 4 nav items: 朝堂(active) / 王朝 / Skills / 旨意
  Right side: bell notification icon + user avatar

LAYER 2 — BREADCRUMB (H:28px):
  "朝堂 · 紫微城"

LAYER 3 — EMPEROR & CROWN PRINCE AREA (H:180px):
  Left (w:50%) — 天堂 (Emperor):
    - Imperial seal icon (48×48px)
    - Title "天堂 · 天子" in serif 24px gold
    - Subtitle stats showing active edict count + online ministers
  Right (w:50%) — 明堂 (Crown Prince):
    - Crown prince icon (48×48px)
    - Title "明堂 · 太子" in serif 20px
    - Subtitle "监国 · 待分拣 N 条旨意"

LAYER 4 — CENTRAL AXIS + THREE DEPARTMENTS (H:360px):
  Center: Central axis visualization (golden light band flowing top to bottom)
    - Light band animation: gold particles flowing southward
  Three department columns (equal width):
    - 中书省 (Left): Blue accent (#3b82f6) — Pending draft count
    - 门下省 (Center): Amber accent (#f59e0b) — Pending review count
    - 尚书省 (Right): Emerald accent (#10b981) — Pending dispatch count

LAYER 5 — SIX MINISTRIES + CHARIOT (H:auto, fill remaining):
  Left (w:60%): Six ministry cards (2×3 grid, 16px gap)
    Each card: ministry name(Chinese) + job title + health status + token count
    6 ministries: 户部/吏部/兵部/刑部/工部/礼部
  Right (w:40%): Six Horses Chariot visualization
    6 horse icons arranged in 2 rows, each with health status
    Horses float with subtle animation

COLORS: bg #0a0e17, accent gold #d4a843, card bg #1a1f2e

ANIMATION:
  - Central axis: golden particles flowing south (3s loop)
  - Six horses: independent floating animation (stagger 0.3s)
  - Seamless entrance for all layers
```

### [Claude Sonnet 4.6] 迭代提示词（六部卡片细化）

```
Iterate on the six ministry cards in Layer 5 of the Court Hall page.

For each ministry card:
- Add a progress/health bar at the bottom
- Show current token balance as a monospaced number
- Add hover state: border glow + slight upward shift (4px)
- Add click state: ink ripple effect

Each ministry card should have 4 Variants:
1. Default — normal operation
2. Hover — gold border glow, slight scale 1.02x, cloud pattern deepens
3. Active/Selected — expanded view showing detailed metrics
4. Error — red border, pulse animation on health bar

Keep all other layers unchanged.
```

## P5：旨意工坊 · 圣旨发令 ⭐ 核心页

**路由**：`/edict/create` | **尺寸**：1440×900
**推荐模型**：Claude Opus（复杂交互闭环 + 动效分镜）

### [Claude Opus] 初始提示词

```
You are a senior UI designer specializing in ancient Chinese imperial document
design and interactive form layouts. Create an "Edict Workshop" page where users
create new imperial decrees.

FRAME: 1440×900, dark theme.

LAYOUT (3 vertical sections):

SECTION 1 — HEADER (H:56px):
  DynastyNavbar + Breadcrumb "旨意板 > 新建旨意"

SECTION 2 — EDICT PREVIEW (Left, w:600px, fixed):
  EdictScroll component (partially unfurled, live preview):
  - Title area: shows typed title in serif font
  - Body area: shows typed content in vertical writing
  - Right margin: jade seal placeholder (shows after submission)
  - Real-time preview updates as user types in form

SECTION 3 — FORM PANEL (Right, w:fill remaining):
  Auto Layout vertical form:

  1. 旨意类型 (Edict Type):
     Tab selector: 敕书(default) / 制书 / 敕牒

  2. 旨意标题 (Title):
     Input field, placeholder "请输入旨意标题..."
     Validation: 2-50 chars, required

  3. 紧急程度 (Urgency):
     Three pill buttons: 普通(gray) / 火急(amber) / 十万火急(red)
     Default: 普通

  4. 指派六部 (Assign Ministry):
     Six checkbox chips: 户/吏/兵/刑/工/礼
     Multi-select, at least 1 required

  5. 旨意详情 (Details):
     Textarea, placeholder "请输入旨意详情..."
     Validation: 10-2000 chars

  6. ACTION BUTTONS:
     [存为草稿] — secondary ghost button
     [奉天承运·颁旨] — primary CTA button (gold gradient bg with seal icon)

FORM VALIDATION:
  - Validate on blur for title and details
  - Validate all on submit
  - Error: red border + shake animation + error message below field
  - Success: green border + check icon

ANIMATION (keyframe descriptions in notes):
  - Edict scroll unfurls as user types (real-time preview)
  - Submit button click → seal stamping animation (1.2s, 5 phases)
  - Form field focus: gold border + slight glow
```

## P8：双星桥 · Family × Dynasty 全景

**路由**：`/bridge` | **尺寸**：1440×900
**推荐模型**：Gemini 3.1 Pro（创意 + 双架构桥接）

### [Gemini 3.1 Pro] 初始提示词

```
You are a senior UI designer creating a "Dual Star Bridge" page that visualizes
the bridge between two architectures: Family (团队人格) and Dynasty (朝代技能).

FRAME: 1440×900, dark theme.

LAYOUT:

SECTION 1 — HEADER (H:56px):
  DynastyNavbar + Breadcrumb "朝堂 > 双星桥"

SECTION 2 — BRIDGE VISUALIZATION (Main area):
  Two circular formations connected by a golden bridge:

  LEFT CIRCLE — "Family · 家" (直径400px, left side):
    12 AI personas arranged in a circular formation
    Each persona: circular avatar (40×40px) with name
    12 personas: 元启·天枢 / 元启·玉衡 / 元启·瑶光 / 元启·开阳 / etc.
    Selected persona glows with gold ring

  RIGHT CIRCLE — "Dynasty · 王朝" (直径400px, right side):
    13 dynasties arranged in a circular formation
    Each dynasty: icon + name
    Selected: gold ring + glow

  CENTER — "星汉桥" (Golden bridge connecting two circles):
    Flowing gold particles from Family to Dynasty
    When a persona is selected, bridge lights up with connecting lines
    to skill-available dynasties

SECTION 3 — DETAIL PANEL (Bottom, H:200px):
  When a Family persona is selected → shows persona details + dynasty skills
  When a Dynasty is selected → shows available personas for that dynasty's skills
  Split panel if both selected: shows compatibility matrix

ANIMATION:
  - Both circles rotate slowly (30s per rotation, opposite directions)
  - Bridge gold particles flow continuously
  - Selected item: pulsing glow ring
  - Connecting lines: dashed gold lines animate from family to dynasty
```
