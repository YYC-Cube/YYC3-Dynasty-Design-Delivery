## P1：欢迎启动页 · 玉玺启宫门

**路由**：`/welcome` | **尺寸**：520×680 弹窗 / 全屏适配

```
You are a senior UI designer specializing in ancient Chinese imperial aesthetics
fused with modern dark-mode dashboards. Create a welcome/launch page for
"YYC³ Dynasty" — an AI multi-agent governance platform.

FRAME: 520×680px, dark theme.

LAYOUT:
1. TOP SECTION (centered):
   - 56×56px 3D imperial jade seal icon, golden with dragon knob
   - Title "YYC³ Dynasty" in serif 28px, gold (#d4a843)
   - Subtitle "三省以治 · 六部以行" in serif 14px, secondary gold

2. MIDDLE SECTION (3×2 grid):
   - Card 1: 朝堂听政 → Enter Court Hall
   - Card 2: 王朝典章 → Dynasty Timeline
   - Card 3: 勋绩考课 → Honor Wall
   - Card 4: 敕令流转 → Edict Board
   - Card 5: 太史监候 → System Monitor
   - Card 6: 宫阙规制 → System Settings
   Card style: bg rgba(212,168,67,0.08), border 1px rgba(212,168,67,0.2), radius 12px

3. BOTTOM SECTION:
   - Golden gradient divider
   - Status text: "紫微城中枢 · 十二朝臣当值"
   - CTA button "奉天承运" (gold gradient bg, serif 16px)

ANIMATION:
   - Jade seal drops from top with red ink stamp spread
```

## P3：十三王朝时间轴 · 洛邑千年

**路由**：`/timeline` | **尺寸**：1440×900

```
FRAME: 1440×900, dark theme.

Create a timeline/selector page showing 13 Chinese dynasties.

LAYOUT:
1. HEADER: DynastyNavbar + Breadcrumb "朝堂 > 十三王朝"

2. LEFT SIDEBAR (Dynasty Selector, w:280px):
   - Vertical list of 13 dynasties (vertical Auto Layout)
   - Each item: dynasty name + reign years + small icon
   - Selected state: gold bg highlight
   - 13 dynasties in order: 夏/商/周/秦/汉/三国/晋/南北朝/隋/唐/宋/明/清

3. MAIN CONTENT (Fill remaining):
   - Hero area: Selected dynasty name (32px, serif) + imperial color
   - Description: 2-3 sentence dynasty summary
   - Skills Grid: 4×N grid of skill cards
   - Each skill card: 200×140px, icon + name + short desc

COLORS: bg #0a0e17, card bg #1a1f2e, gold accent #d4a843
Each dynasty has its own accent color (e.g., Tang gold, Han azure)
```

## P6：旨意板 · 敕令流转监控

**路由**：`/edict` | **尺寸**：1440×900

```
FRAME: 1440×900, dark theme.

Create an edict tracking board page.

LAYOUT:
1. HEADER: DynastyNavbar + floating action button "新建旨意"(gold gradient, rounded-full)
2. Filter bar: search input + status filter dropdown + sort toggle
3. Edict list (vertical Auto Layout):
   Each item:
   - Left: Edict ID (mono font, gold)
   - Center: Title, urgency badge, created time
   - Right: EdictStepBar (7-stage progress, compact horizontal variant)
   - Bottom: current stage label + assigned ministry tags
4. Bottom: Pagination [1][2][3]...[Next]

INTERACTION: Click item → open P9 detail
Hover: gold border + slight scale

COMPONENTS: DynastyNavbar, EdictStepBar (compact), Pagination
```

## P9：敕令详情 · 圣旨阅览

**路由**：`/edict/:id` | **尺寸**：1440×900

```
Create an edict detail page showing a complete imperial decree.

FRAME: 1440×900, dark theme.

LAYOUT:
1. HEADER: DynastyNavbar + Breadcrumb "旨意板 > 敕令详情"
2. LEFT PANEL (w:600px): Imperial edict scroll display
   - EdictScroll component (unfurled state, 600px width)
   - Edict content in vertical writing mode (right to left)
   - Bottom: Imperial seal stamp (Stamped state)
3. RIGHT PANEL (fill remaining):
   - TOP: EdictStepBar (7-stage, vertical orientation, full width)
   - MIDDLE: SealChain component (showing all signing seals)
   - BOTTOM: Flow log timeline (reverse chronological)
     Each entry: timestamp + operator name + action + department badge
4. ACTION BAR: [催办] [撤回] [归档] [打印] buttons

STATES: Loading (skeleton) / Loaded / Error (edict not found "此旨已佚")

COMPONENTS: DynastyNavbar, EdictScroll, EdictStepBar, SealChain
```

## P10：太史监候 · 系统监控台

**路由**：`/monitor` | **尺寸**：1440×900

```
Create a system monitoring dashboard styled as an astronomical observatory.

FRAME: 1440×900, dark theme.

LAYOUT:
1. HEADER: DynastyNavbar + Breadcrumb "朝堂 > 太史监候"
2. TOP ROW (4 metric cards, equal width):
   - Metric 1: "当值朝臣" Agent Online Count (mono font 32px, gold)
   - Metric 2: "敕令流转" Active Edicts Count
   - Metric 3: "令牌存量" Token Balance
   - Metric 4: "朝堂运转" System Uptime

3. MIDDLE SECTION (2-column):
   - LEFT (w:50%): "天子驾六" 6 horse status panel
     Each horse: name + health bar + current token count
     3 health states: Normal(green) / Warning(amber) / Error(red)
   - RIGHT (w:50%): "三省令牌" token usage chart
     Bar chart showing token usage by ministry

4. BOTTOM SECTION: "天象日志" real-time event log
   - Auto-scrolling log feed (latest at top)
   - Each entry: timestamp + event type badge + description
   - Event types: [敕令][令牌][告警][系统]

COMPONENTS: DynastyNavbar, MetricCard, HorseStatusBar, TokenChart, LogFeed
```
