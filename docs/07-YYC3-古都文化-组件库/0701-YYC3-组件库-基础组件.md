# 基础组件

## 1. DynastyNavbar（全局导航栏）

| 属性 | 值 |
|------|-----|
| 高度 | 56px（固定） |
| 背景 | `rgba(10,14,23,0.95)` + `backdrop-filter: blur(12px)` |
| 布局 | 左：Logo + 4主导航项 / 右：搜索+通知+头像 |
| 导航项 | 朝堂 / 王朝 / Skills / 旨意 |
| 通知角标 | 朱砂红圆形徽章，数字最大 "99+" |
| 使用页面 | 所有页面 |

**状态：** Default / Scrolled（加底部阴影）/ Active（当前页高亮）

---

## 2. Breadcrumb（面包屑）

| 属性 | 值 |
|------|-----|
| 高度 | 28px |
| 分隔符 | ">" 或 回纹分隔符 |
| 文字 | `--color-text-secondary` `#9ca3af` |
| 当前页 | `--color-text-primary` `#e5e7eb` |
| 使用页面 | 所有页面（P1除外） |

---

## 3. SideQuickEntry（侧边快捷入口）

| 属性 | 值 |
|------|-----|
| 宽度 | 48px |
| 位置 | 页面右侧，底部 |
| 图标 | 4-5 个垂直排列：朝堂/旨意/双星桥/Hub/返回 |
| 动效 | hover 时展开显示文字标签 |
| 使用页面 | P2 专用 |

---

## 4. Toast Notification（通知提示）

| 属性 | 值 |
|------|-----|
| 位置 | 右上角 (top: 80px, right: 24px) |
| 动画 | 从右滑入 (400ms ease-out) |
| 自动消失 | P3:3s / P2:5s / P1:8s / P0:手动 |
| 堆叠 | 最多可见 3 条 |
| 队列 | 最大 50 条，超限淘汰最旧 P3 |

---

## 5. SearchPalette（全局搜索面板）

| 属性 | 值 |
|------|-----|
| 唤起 | `⌘K` 快捷键 |
| 尺寸 | 520×480px |
| 过滤 | 分类过滤（敕令/朝代/Skill/朝臣） |
| 导航 | `↑↓` 键盘导航，`Enter` 跳转 |
| 支持 | 拼音/中文/英文 模糊匹配 |

---

## 6. 表单组件

| 组件 | 尺寸 | 状态 | 使用 |
|------|------|------|------|
| DynastyInput | H:44px | Default/Focus/Valid/Error/Disabled | P5 标题输入 |
| DynastySelect | H:44px | Default/Open/Selected | P5 类型选择 |
| DynastyTextarea | H:200px | Default/Focus/Valid/Error | P5 详情输入 |
