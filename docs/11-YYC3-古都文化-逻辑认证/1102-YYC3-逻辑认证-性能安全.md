# 性能与安全规范

## 一、性能预算

| 指标 | 目标值 | 测量工具 |
|------|--------|---------|
| FCP (First Contentful Paint) | < 1.5s | Lighthouse |
| LCP (Largest Contentful Paint) | < 2.5s | Lighthouse |
| TBT (Total Blocking Time) | < 200ms | Lighthouse |
| CLS (Cumulative Layout Shift) | < 0.1 | Lighthouse |
| TTI (Time to Interactive) | < 3.5s | Lighthouse |
| 首页 JS Bundle | < 150KB (gzip) | Webpack Analyzer |
| 图片总大小 | < 500KB (首页) | Network Tab |

**优化策略：**
- 路由懒加载：11 页均使用 dynamic import
- 图片优化：WebP 格式 + 响应式 srcset
- 字体子集化：提取仅用到的古文字符
- SSG 预渲染：P1/P2/P3 首屏静态生成
- ISR 增量再生：P6 旨意板每 60s 重新验证

## 二、安全规范

**五层防护：**

| 层级 | 防护 | 措施 |
|:----:|:----:|------|
| L1 | XSS 防护 | DOMPurify 清洗 + CSP 头部 + 禁用 dangerouslySetInnerHTML |
| L2 | CSRF 防护 | 玉玺 Token (JWT) + SameSite=Strict Cookie + 关键操作二次确认 |
| L3 | 权限控制 | RBAC 5级 + 路由守卫 + API 鉴权 + 操作审计 |
| L4 | 数据加密 | TLS 1.3 + AES-256-GCM（敏感字段）+ 玉玺JWT含签名+过期 |
| L5 | 依赖安全 | pnpm audit 每周 + Dependabot + 锁定 pnpm-lock.yaml |
