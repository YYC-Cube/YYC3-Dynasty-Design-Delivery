# 测试与 API 契约

## 一、API 契约

| 组 | 端点 | 说明 |
|:--:|------|------|
| 敕令 | `GET/POST/PATCH/DELETE /api/edicts/:id` | CRUD |
| 流转 | `POST /api/edicts/:id/advance` (推进) / `.../veto` (封驳) | 推进/驳回 |
| 六部 | `GET /api/ministries/health` / `.../metrics` | 健康/指标 |
| 通知 | `GET /api/notifications` / `PATCH .../:id/read` / `POST .../read-all` | 列表/已读 |
| 认证 | `POST /api/auth/verify-seal` / `.../refresh` | 玉玺验证/刷新 |
| 搜索 | `GET /api/search?q={query}&type={type}` | 全局搜索 |

**错误响应格式：**
```json
{ "error": { "code": "EDICT_NOT_FOUND", "message": "此旨已佚" } }
```
