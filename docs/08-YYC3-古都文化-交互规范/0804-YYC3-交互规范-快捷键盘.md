# 键盘快捷键与右键菜单

## 全局快捷键

| 快捷键 | 功能 | 作用域 |
|--------|------|--------|
| `⌘K` / `Ctrl+K` | 全局搜索 | 全局 |
| `⌘N` / `Ctrl+N` | 新建旨意 | 全局 |
| `⌘1` | 跳转 P1 启动页 | 全局 |
| `⌘2` | 跳转 P2 朝堂大厅 | 全局 |
| `⌘3` | 跳转 P3 十三王朝 | 全局 |
| `⌘4` | 跳转 P4 勋章墙 | 全局 |
| `⌘5` | 跳转 P5 旨意工坊 | 全局 |
| `⌘6` | 跳转 P6 旨意板 | 全局 |
| `⌘7` | 跳转 P7 Hub | 全局 |
| `⌘8` | 跳转 P8 双星桥 | 全局 |
| `Escape` | 关闭弹窗/面板 | 弹窗内 |
| `⌘Z` / `Ctrl+Z` | 撤销（表单内） | P5 表单 |
| `⌘⇧Z` / `Ctrl+Shift+Z` | 重做 | P5 表单 |
| `⌘S` / `Ctrl+S` | 保存草稿 | P5 表单 |
| `⌘Enter` / `Ctrl+Enter` | 提交旨意 | P5 表单 |
| `⌘B` / `Ctrl+B` | 批量选择模式 | P6 旨意板 |
| `Space` | Hub 语音输入 | P7 Hub |
| `⌘/` | 显示快捷键帮助 | 全局 |
| `⌘F` / `Ctrl+F` | 筛选面板 | P6 旨意板 |

## 右键菜单（Context Menu）

在 P6 旨意板卡片上右键：

```
┌──────────────────┐
│ 📋 查看详情       │
│ 📝 复制敕令编号   │
│ ⚡ 催办           │
│ ─────────────── │
│ 🔗 复制链接       │
│ 📊 查看流转日志   │
└──────────────────┘
```

## Tab 键导航顺序

```
1. DynastyNavbar → 主导航项 (L→R)
2. 面包屑 → 当前页路径
3. 页面主体 → 按 DOM 自然顺序
4. 侧边快捷入口 → 上→下
5. Footer / 通知中心
```

## 焦点指示器

| 属性 | 值 |
|------|-----|
| 样式 | `2px solid #d4a843`（鎏金） |
| 偏移 | `outline-offset: 2px` |
| 圆角 | 继承元素 border-radius |
| 可见性 | `:focus-visible` 伪类 |
| 触发 | 仅限于键盘导航，鼠标点击不显示 |
