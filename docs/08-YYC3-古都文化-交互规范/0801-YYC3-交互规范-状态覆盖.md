# 全局状态覆盖（四态标准）

## 四态标准

每个可交互组件必须覆盖以下四种状态：

| 状态 | 触发 | 视觉表现 |
|------|------|---------|
| **Default** | 初始 / 无交互 | 标准外观，符合 Design Token |
| **Hover** | 鼠标悬停 | 鎏金边框微亮 + 上浮 2-4px + 云纹加深 |
| **Active** | 鼠标按下 / 选中 | 印泥色波纹扩散 + 缩小 0.97x |
| **Disabled** | 不可操作 | 50% 透明度 + 无交互反馈 |

## 附加状态

| 状态 | 触发 | 视觉表现 |
|------|------|---------|
| **Focus** | 键盘 Tab 导航聚焦 | 鎏金 2px outline + offset 2px |
| **Loading** | 数据请求中 | Skeleton 骨架屏 / 龙纹旋转加载器 |
| **Error** | 操作失败 / 校验未通过 | 朱砂红边框 + 抖动动画 + 错误提示 |
| **Empty** | 无数据 | 空状态插画 + 引导文字 |
| **Selected** | 多选模式选中 | 金色边框 + 左侧复选框 ✓ |
| **Locked** | 权限不足 | 灰白半透明 + 🔒 图标覆盖 |

## 交互规范

| 场景 | 属性 | 值 |
|------|------|-----|
| 卡片点击 | 光标 | `cursor: pointer` |
| 按钮点击 | 反馈 | 缩小 0.95x + 波纹扩散（150ms） |
| 输入聚焦 | 边框 | 鎏金 `1px solid #d4a843` + 微光晕 |
| 文本选择 | 颜色 | 背景 `rgba(212,168,67,0.2)` |
| 拖拽 | 光标 | `grab` / `grabbing` |
| 滚动条 | 样式 | 鎏金薄条，`8px` 宽，圆角 `4px` |
| 过渡 | 缓动 | `cubic-bezier(0.4, 0, 0.2, 1)` |
| 过渡 | 时长 | `300ms`（通用），`150ms`（快速） |
