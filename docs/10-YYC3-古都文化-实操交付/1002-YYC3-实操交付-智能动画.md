# Figma Smart Animate 参数详表

## 一、玉玺盖章动效

| 参数 | 值 |
|------|-----|
| Trigger | On Click |
| Duration | 1200ms |
| Easing | `cubic-bezier(0.34, 1.56, 0.64, 1)` |
| Match | Layer Name |

**关键帧：**
| State | Y | Opacity | Scale | Note |
|:-----:|:-:|:-------:|:-----:|------|
| Idle | 0 | 0.6 | 1 | 微光呼吸 |
| Hover | -8px | 1 | 1.05 | 螭虎钮抬起 |
| Pressed | +8px | 1 | 1.05 | 印泥波纹 |
| Stamped | 0 | 1 | 1 | 印文浮现 |

## 二、圣旨展开动效

| 参数 | 值 |
|------|-----|
| Duration | 800ms |
| Easing | `cubic-bezier(0, 0, 0.2, 1)` |
| Match | Layer Name |

**关键帧：**
| State | Width | Opacity |
|:-----:|:-----:|:-------:|
| Collapsed | 60px | 0.3 |
| Unfurling | 200px | 0.7 |
| Expanded | 600px | 1 |

## 三、页面过渡动效

| 参数 | 值 |
|------|-----|
| Duration | 400ms |
| Easing | Ease In and Out |

**过渡模式：**
| 场景 | 动效 |
|------|------|
| P1→P2 | 玉玺放大填满→淡出露出朝堂（800ms） |
| 页面间 | 滑动推入（Slide In Right → Left） |
| 弹窗 | 缩放上浮（Scale 0.8→1, Opacity 0→1） |
| 详情页 | 从列表项位置展开（Shared Element Transition） |

## 四、天子驾六状态机动画

| 切换 | 时长 | 动效 |
|:----:|:----:|------|
| N→W | 300ms | 琥珀渐变 + 马蹄微颤 |
| W→E | 500ms×2 | 红色脉冲 + 绳索闪烁 |
| E→N | 800ms | 金色粒子修复 |
| 全局刷新 | 600ms | 6马重置 stagger 0.1s |
