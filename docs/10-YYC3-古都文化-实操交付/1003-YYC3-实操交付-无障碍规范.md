# 无障碍 (A11y) 标注规范

## 一、色彩对比度（WCAG 2.1 AA）

| 元素组合 | 前景色 | 背景色 | 对比度 | 等级 |
|---------|--------|--------|:------:|:----:|
| 正文/玄墨底 | `#e5e7eb` | `#0a0e17` | 13.5:1 | AAA |
| 辅助文字/玄墨底 | `#9ca3af` | `#0a0e17` | 6.2:1 | AA |
| 鎏金标题/玄墨底 | `#d4a843` | `#0a0e17` | 8.3:1 | AAA |
| 朱砂警告/玄墨底 | `#c2414b` | `#0a0e17` | 5.1:1 | AA |
| 石青链接/靛青底 | `#3b82f6` | `#1a1f2e` | 4.8:1 | AA-large |

## 二、焦点指示器

| 属性 | 值 |
|------|-----|
| 样式 | `2px solid #d4a843`（鎏金） |
| 偏移 | `outline-offset: 2px` |
| 圆角 | 继承元素 border-radius |
| 可见性 | `:focus-visible` 伪类 |

## 三、ARIA 标注

```html
<!-- DynastyNavbar -->
<nav aria-label="朝堂导航">
  <ul role="menubar">
    <li role="menuitem" aria-current="page">朝堂</li>
  </ul>
</nav>

<!-- EdictStepBar -->
<div role="progressbar" aria-valuenow="3" aria-valuemin="1"
     aria-valuemax="7" aria-label="敕令流转进度: 第3阶段">

<!-- MinistryCard -->
<article aria-label="户部运行状态: 正常">
  <span class="sr-only">户部: 度支正常, 赋税已入库</span>
</article>

<!-- Toast P0 -->
<div role="alert" aria-live="assertive" aria-atomic="true"></div>
<!-- Toast P1-P3 -->
<div role="status" aria-live="polite" aria-atomic="true"></div>
```

## 四、可读性规范

| 规范 | 要求 |
|------|------|
| 最小正文字号 | 14px |
| 最小可交互元素尺寸 | 44×44px |
| 最小行高 | 1.5 |
| 最大行宽 | 75ch |
| 段落间距 | 16px |
| 文字缩放 | 支持 200% 不截断 |
