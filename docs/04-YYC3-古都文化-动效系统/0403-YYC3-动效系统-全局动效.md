# 全局动效清单

## 一、页面级动效

| 动效 | 时长 | 缓动 | 说明 |
|------|------|------|------|
| 朝堂入场 | 1.2s | ease-out | 首次加载：宫门开启→云纹散开→内容淡入 |
| 页面切换 | 400ms | ease-in-out | 旧页墨色晕染消失 + 新页从卷轴展开 |
| 数据刷新 | 600ms | ease | 数字滚动变化 + 微光扫过卡片 |
| 弹窗进场 | 300ms | ease-out | 缩放上浮 (Scale 0.8→1, Opacity 0→1, Y +20→0) |
| 详情展开 | 500ms | ease-out | 从列表项位置展开（Shared Element Transition） |

## 二、组件级动效

| 动效 | 时长 | 缓动 | 说明 |
|------|------|------|------|
| 三省流转光点 | 3s 循环 | linear | 鎏金粒子沿中轴线从北向南流动 |
| 六部卡片 hover | 300ms | ease-out | 边框金线爬行一圈 + 微上浮 4px + 云纹加深 |
| 勋章 hover | 400ms | ease-out | 勋章旋转 8° + 星标闪烁 |
| 按钮 hover | 200ms | ease-out | 鎏金渐变更亮 + 微放大 1.02x |
| 按钮 click | 150ms | ease-in | 缩小 0.95x + 印泥色波纹扩散 |
| Toast 通知 | 400ms | ease-out | 从右滑入 + 微弹 + 5s 后淡出 |
| 列表项新增 | 500ms | ease-out | 从上方降入 + 金线高亮 1.5s 后消退 |
| Loading | ∞ | linear | 龙纹逐段流光循环（3s/周期） |
| 云纹漂移 | 30s 循环 | linear | 极缓慢水平移动 |
| 天子驾六悬浮 | 3-5s 循环 | ease-in-out | 每匹马 ±6px 独立悬浮，stagger 0.3s |

## 四、页面过渡动效参数

```

  ├─ Duration: 400ms
  ├─ Easing: Ease In and Out
  └─ 过渡模式:
      ├─ P1→P2: 玉玺放大填满屏幕 → 淡出露出朝堂 (800ms)
      ├─ 页面间: 滑动推入 (Slide In Right → Left)
      ├─ 弹窗: 缩放上浮 (Scale 0.8→1, Opacity 0→1, Y +20→0)
      └─ 详情页: 从列表项位置展开 (Shared Element Transition)
```

## 五、天子驾六状态机动画

| 切换方向 | 时长 | 动效描述 |
|---------|------|---------|
| Normal → Warning | 300ms | 马身颜色渐变（琥珀化），马蹄微颤 |
| Warning → Error | 500ms × 2次 | 马身红色脉冲，绳索闪烁 |
| Error → Normal | 800ms | 马身颜色恢复，金色粒子修复 |
| 全局刷新 | 600ms | 六马同时重置，stagger 0.1s |
