# 玉玺盖章动效（The Sealing Effect）

## 组件设计

- 正方形玉玺主体（80×80px），温润白玉材质
- 上雕螭虎钮（龙形把手），鎏金描边
- 印文："受命于天·既寿永昌"（九叠篆，8字2行）
- 投影：`box-shadow: 0 8px 32px rgba(212,168,67,0.3)`

## 五变体（Variants）

| Variant | 状态 | 视觉效果 | 动画时长 |
|---------|------|---------|---------|
| Idle | 待命 | 静态，微光呼吸 | 持续 |
| Hover | 悬停 | 放大 1.1x + 边框发光 + 螭虎钮抬起 | 300ms |
| Pressing | 盖章中 | 向下位移 8px + 印泥色扩散波纹 | 200ms |
| Stamped | 已盖章 | 回弹 + 金色粒子散开 + 印文浮现 | 400ms |
| Locked | 无权限 | 灰白半透明 + 🔒 覆盖 | — |

## Figma Smart Animate 配置

| 参数 | 值 |
|------|-----|
| Trigger | On Click（按钮点击） |
| Action | Navigate to → "Sealing Frame" |
| Animation | Smart Animate |
| Duration | 1200ms |
| Easing | Custom cubic-bezier(0.34, 1.56, 0.64, 1) |
| Match | Layer Name（玉玺组件各层命名一致） |

## 动画分镜（1.2s 总时长）

```
Phase 1 — 玉玺升起 (0-300ms)
  ├─ 玉玺从底部弹入 (ease-out-back)
  ├─ 半透明虚影逐渐实化
  └─ 螭虎钮龙头朝前，玉玺角度微倾 5°

Phase 2 — 悬停蓄力 (300-500ms)
  ├─ 玉玺悬停在目标上方 20px 处
  ├─ 印泥色光环从中心扩散 (朱砂红, opacity 0→0.4→0)
  └─ 周围金色粒子聚拢

Phase 3 — 落印 (500-700ms)
  ├─ 玉玺快速下压至目标位置 (ease-in, cubic-bezier)
  ├─ 目标区域微微凹陷 (transform: scale(0.97))
  ├─ 金色 shockwave 波纹从落点扩散
  └─ 印泥溅射粒子 (8-12个, 朱砂红)

Phase 4 — 玉玺回位 (700-900ms)
  ├─ 玉玺弹回原位 (ease-out-bounce)
  ├─ 金色粒子消散
  └─ 印文浮现：九叠篆从模糊到清晰

Phase 5 — 印记永驻 (900-1200ms)
  ├─ 朱砂红印文完全清晰
  ├─ 印章边框金色微光呼吸 (3次, 每次500ms)
  └─ 最终定格为静态印章
```

## 关键帧状态

| 状态 | Y轴 | Opacity | Scale | 其他 |
|------|:---:|:-------:|:-----:|------|
| Idle | 0 | 0.6 | 1 | — |
| Hover | -8px | 1 | 1.05 | 边框发光 |
| Pressed | +8px | 1 | 1.05 | 印泥波纹扩散 |
| Stamped | 0 | 1 | 1 | 印文浮现 + 粒子 |

## 粒子系统参数

| 属性 | 值 |
|------|-----|
| 粒子数量 | 8-12 个 |
| 颜色 | 朱砂红 `#c2414b` + 鎏金 `#d4a843` |
| 运动轨迹 | 径向扩散 + 上浮 |
| 大小 | 2-6px（随机） |
| 寿命 | 400-800ms（随机） |
| 缓动 | ease-out |
