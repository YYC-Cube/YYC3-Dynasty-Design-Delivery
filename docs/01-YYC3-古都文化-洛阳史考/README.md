# 01-YYC3-古都文化-洛阳史考

## 目录概述

本目录聚焦 **YYC³ Dynasty** 项目的文化根基层，涵盖洛阳紫微城中轴线空间映射、12组文化符号体系及四类权力载体的系统功能映射。是整站设计的文化底层与语义体系。

## 核心数据速查

```
文化符号总数:     12 组
紫微城节点数:     8 节点（天堂→明堂→应天门→天津桥→端门→定鼎门）
权力载体类型:     4 类（玉玺/圣旨/竹箴/鱼符）
玉玺等级:         5 级（传国/行玺/信玺/之玺/天子玺）
圣旨类型:         5 类（制书/敕书/敕牒/堂帖/密奏）
鱼符部门:         6 部（户/吏/兵/刑/工/礼）
```

## 设计原则

1. **文化为体，功能为用**：所有符号须有明确系统落点
2. **空间即权力**：中轴线方位对应系统权限层级
3. **载体即协议**：古物形态映射消息协议与执行凭证
