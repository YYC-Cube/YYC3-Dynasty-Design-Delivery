# 文化符号体系

## 概述

以下 12 组文化符号贯穿全部 11 个页面，**每一组均有明确的系统功能落点**，不做纯装饰。符号是用户理解系统语义的第一通道。

## 符号设计规范

### 1. 传国玉玺

- 尺寸：80×80px（标准），40×40px（缩略）
- 颜色：白玉底座 + 鎏金描边 + 朱砂印文
- 印文："受命于天·既寿永昌"（九叠篆，8字2行）
- 状态：Idle / Hover / Pressing / Stamped / Locked（5变体）

### 2. 圣旨卷轴

- 底色：明黄丝绢 `#f5e6c8`，渐变做旧纹理
- 镶边：金色龙纹织锦（宽16px）
- 天头地头：蓝色绫绢（天40px/地30px）
- 轴头：深檀木色 `#4a3728` + 鎏金包边

### 3. 五爪金龙

- 使用场景：Hero背景暗纹、玉玺表面、Loading动画
- 透明度：`rgba(212,168,67,0.06)`（背景），`opacity:1`（Loading）
- 动效：龙纹流光——轮廓线从龙头到龙尾逐段点亮（3s循环）

### 4. 祥云纹

- 样式：如意云头 + 流云尾
- 透明度：`rgba(212,168,67,0.04)`
- 动效：极缓慢水平漂移（30s循环），营造"风云流动、朝堂常新"

### 5. 回纹

- 样式：连续T形转折
- 颜色：鎏金色 `rgba(212,168,67,0.3)`
- 尺寸：2px 宽描边

### 6. 竹简

- 样式：竖向竹片排列，每片宽约 12px
- 颜色：宣纸旧色 `#1E180E` + 金色编绳
- 动效：展开时竹片逐片翻开（stagger 50ms）

### 7. 宫灯

- 尺寸：24×32px（导航栏用）
- 颜色：鎏金灯架 + 暖黄光晕 `rgba(212,168,67,0.3)`
- 状态：点亮/熄灭/闪烁（3状态）

### 8. 朱砂印泥

- 颜色：`#c2414b`（朱砂红）
- 透明度：盖印初期 opacity 0.6 → 干燥后 opacity 1
- 纹理：轻微飞白/散开效果，非纯色填充

### 9. 十二章纹

- 12种纹章：日/月/星辰/山/龙/华虫/宗彝/藻/火/粉米/黼/黻
- 尺寸：16×16px（勋章徽标）
- 颜色：鎏金单色描边

### 10. 汉砖纹

- 样式：错缝排列矩形砖块
- 颜色：深黛底色 `#1a1f2e` + 极细金线 `rgba(212,168,67,0.15)` 分隔
- 使用区域：Footer 背景全幅

### 11. 海水江崖

- 样式：层叠波浪 + 中央耸立山崖
- 颜色：蓝金渐变色（石青 → 鎏金）
- 使用区域：侧栏底部全幅装饰

### 12. 风铃

- 尺寸：16×16px（图标级）
- 样式：三线风铃 + 铃舌
- 状态：静止（常规）/ 摇动（新通知）/ 急促（P0告警）
