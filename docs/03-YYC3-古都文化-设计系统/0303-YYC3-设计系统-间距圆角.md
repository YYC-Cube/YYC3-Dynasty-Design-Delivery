# 间距 & 圆角 & 阴影

## 间距体系

间距遵循 4px 基准单位递增：

| Token 名 | 值 | 场景 |
|----------|-----|------|
| `--spacing-xs` | 4px | 微间距、图标与文字间距 |
| `--spacing-sm` | 8px | 紧凑间距、内边距 |
| `--spacing-md` | 16px | 标准间距、卡片内边距、元素间距 |
| `--spacing-lg` | 24px | 大间距、区块间距、页面边距 |
| `--spacing-xl` | 32px | 特大间距、段落间距 |
| `--spacing-2xl` | 48px | 超大间距、页面顶部/底部 |
| `--spacing-3xl` | 64px | 极致间距、大片内容间隔 |

**布局参考：**
- 页面内边距：`--spacing-lg`（24px）
- 卡片内边距：`--spacing-md`（16px）
- 卡片间距：`--spacing-md`（16px）
- 列表项间距：`--spacing-sm`（8px）
- 文字与图标间距：`--spacing-xs`（4px）

## 圆角体系

| Token 名 | 值 | 场景 |
|----------|-----|------|
| `--radius-none` | 0px | 无圆角（特殊装饰元素） |
| `--radius-sm` | 4px | 小圆角（输入框内部、标签） |
| `--radius-md` | 8px | 中圆角（按钮、弹窗） |
| `--radius-lg` | 12px | 大圆角（卡片、面板） |
| `--radius-xl` | 16px | 特大圆角（大型弹窗、Modal） |
| `--radius-full` | 9999px | 全圆角（徽章、头像、标识） |

**应用矩阵：**

| 元素 | Token | 值 |
|------|-------|-----|
| 卡片（Card） | `--radius-card` | 12px |
| 按钮（Button） | `--radius-button` | 8px |
| 头像/徽章（Avatar） | `--radius-full` | 9999px |
| 输入框（Input） | `--radius-sm` | 4px |
| 弹窗（Modal） | `--radius-xl` | 16px |
| 通知（Toast） | `--radius-md` | 8px |
| 下拉面板（Panel） | `--radius-lg` | 12px |

## 阴影体系

| Token 名 | 值 | 场景 |
|----------|-----|------|
| `--shadow-sm` | `0 1px 3px rgba(0,0,0,0.4)` | 卡片微阴影、普通元素 |
| `--shadow-md` | `0 4px 12px rgba(0,0,0,0.5)` | 面板、下拉菜单 |
| `--shadow-lg` | `0 8px 32px rgba(212,168,67,0.3)` | 玉玺、重要卡片、弹窗 |
| `--shadow-xl` | `0 16px 48px rgba(0,0,0,0.6)` | Modal、大弹窗 |
| `--shadow-glow` | `0 0 20px rgba(212,168,67,0.4)` | 光晕效果、hover强调态 |

**应用矩阵：**

| 元素 | Token |
|------|-------|
| 普通卡片 | `--shadow-sm` |
| 导航栏 | `--shadow-md` |
| 玉玺组件 | `--shadow-lg` |
| 弹窗/Modal | `--shadow-xl` |
| 按钮 hover | `--shadow-glow` |
| 通知 Toast | `--shadow-md` |
| 搜索面板 | `--shadow-xl` |

## 边框体系

| Token 名 | 值 | 场景 |
|----------|-----|------|
| `--border-default` | `1px solid rgba(212,168,67,0.12)` | 默认卡片/面板边框 |
| `--border-selected` | `1px solid #d4a843` | 选中态/激活态边框 |
| `--border-stroke` | `1px solid rgba(212,168,67,0.2)` | 更强可见度边框 |
| `--border-thin` | `1px solid rgba(212,168,67,0.06)` | 极弱分割线 |
