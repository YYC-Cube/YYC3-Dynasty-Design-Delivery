# 03-YYC3-古都文化-设计系统

## 目录概述

本目录定义 YYC³ Dynasty 的完整 Design Token 体系，包含色彩、字体、间距/圆角/阴影、装饰纹样四大模块。是整站视觉一致性的基础规范。

## 子文件清单

| 编号 | 文件名称 | 核心内容 |
|------|---------|---------|
| 0301 | 色彩体系 | 14色主色板 + 4语义色 + 部门色彩 + WCAG AA 对比度 |
| 0302 | 字体规范 | 8级字体体系 + 字体家族 + 圣旨竖排规范 |
| 0303 | 间距圆角 | 7级间距 + 6级圆角 + 5级阴影 + 边框体系 |
| 0304 | 装饰纹样 | 云纹/回纹/龙纹/汉砖纹/海水江崖 五类纹样规范 |

## 核心数据速查

```
色彩: 主色14种 + 语义色4种 + 部门色9种
字体: 8级字号（48px→12px），3个字体家族
间距: 7级（4px→64px），4px 基准
圆角: 6级（0px→9999px）
阴影: 5级（sm→glow）
纹样: 5类（云纹/回纹/龙纹/汉砖纹/海水江崖）
```

## 设计原则

1. **文化为底色**：传统色谱结合现代暗色主题
2. **语义即色彩**：每个色彩都有明确的语义功能
3. **Token 即契约**：所有颜色/字体/间距/圆角/阴影均为 Token 变量
