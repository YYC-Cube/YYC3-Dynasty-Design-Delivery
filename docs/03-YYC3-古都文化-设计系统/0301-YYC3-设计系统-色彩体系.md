# 色彩体系（朝堂色谱）

## 设计理念

以唐代宫廷色谱为灵感基底，融合现代暗色主题 UI 规范。**玄墨为底，鎏金为骨，朱砂为血**——底色深沉厚重，强调色皇家贵气，警示色忠烈醒目。

---
