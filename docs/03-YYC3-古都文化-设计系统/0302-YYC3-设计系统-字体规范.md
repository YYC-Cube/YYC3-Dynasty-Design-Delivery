# 字体体系

## 字体 Token 定义

### 字体家族

| Token 名 | 字体栈 | 使用场景 |
|----------|--------|---------|
| `--font-imperial` | 方正古隶 / 思源宋体 Heavy, serif | 页面主标题、御笔文字 |
| `--font-edict-title` | 思源宋体 Bold, serif | Hero 标题、圣旨标题 |
| `--font-section-title` | 思源宋体 SemiBold, serif | 区块标题、卡片标题 |
| `--font-body` | Inter / SF Pro, sans-serif | 正文内容、表单文字 |
| `--font-caption` | Inter Medium, sans-serif | 标签、辅助信息、时间戳 |
| `--font-seal` | 九叠篆（自定义SVG） | 玉玺印章文字、印文 |
| `--font-mono` | SF Mono / JetBrains Mono Bold, monospace | 数字指标、代码片段 |
| `--font-ancient-quote` | 楷体 / STKaiti Italic, serif | 古风引文、圣旨正文 |

### 字号与字重

| Token 名 | 大小 | 字重 | 古称 | 场景 |
|----------|------|------|------|------|
| `--font-size-yu-bi` | 48px | 900 | 御笔 | 页面主标题（P2朝堂、P1标题） |
| `--font-size-zhao-shu` | 32px | 700 | 诏书 | Hero标题（P3王朝名、P4勋章） |
| `--font-size-zou-zhang` | 20px | 600 | 奏章 | 区块标题（卡片标题、弹窗标题） |
| `--font-size-body` | 14px | 400 | 朝议 | 正文内容 |
| `--font-size-caption` | 12px | 500 | 注解 | 辅助文字、时间戳、标签 |
| `--font-size-seal` | 24px | — | 印篆 | 玉玺印文（九叠篆SVG） |
| `--font-size-mono` | 16px | 700 | 计数 | 数字指标、令牌数量 |

### 行高与字距

| Token 名 | 值 | 场景 |
|----------|-----|------|
| `--line-height-body` | 1.5 | 正文（确保可读性） |
| `--line-height-heading` | 1.2 | 标题 |
| `--letter-spacing-display` | 0.05em | 标题字距（御笔/诏书） |
| `--letter-spacing-body` | 0.02em | 正文字距 |

## 圣旨正文排版规范

圣旨（P5/P9）正文使用**竖排从右向左**布局：

| 属性 | 值 |
|------|-----|
| 方向 | `writing-mode: vertical-rl` |
| 字体 | `--font-ancient-quote`（楷体） |
| 大小 | 16px |
| 行高 | 1.8（竖排宽松） |
| 颜色 | `--color-text-paper` `#E8D5A3` |

## 可读性规范

| 规范 | 要求 |
|------|------|
| 最小正文字号 | 14px |
| 最小可交互元素尺寸 | 44×44px（iOS 标准） |
| 最小行高 | 1.5 |
| 最大行宽 | 75ch（约 600px at 14px） |
| 段落间距 | 16px |
| 文字缩放支持 | 浏览器默认缩放至 200% 不截断 |
