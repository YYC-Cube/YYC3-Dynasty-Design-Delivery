# 装饰纹样库

## 概述

五类中国传统纹样融入 UI 界面，**不喧宾夺主，但见微知著**。每类纹样均有明确的使用场景与透明度/动画规范。

---

## 10.1 云纹（Cloud Pattern）

| 属性 | 值 |
|------|-----|
| 样式 | 如意云头 + 流云尾 |
| 颜色 | `rgba(212,168,67,0.04)` |
| 使用场景 | 全局背景底纹、Hero 区域、卡片 hover 态 |
| 动效 | 极缓慢水平漂移（30s 循环） |
| 意境 | "风云流动、朝堂常新" |

**设计规范：**
- SVG 路径：C 形卷云，三云头 + 一云尾
- 大小：200×120px（单元云纹）
- 排列：不规则散布，无重复单元感
- 透明度随层级增加而加深（背景0.04 → hover 0.08）

---

## 10.2 回纹（Meander Pattern）

| 属性 | 值 |
|------|-----|
| 样式 | 连续 T 形转折（雷纹变体） |
| 颜色 | `rgba(212,168,67,0.3)` 描边 |
| 线宽 | 2px |
| 使用场景 | 卡片边框、分割线、Tab 激活态下划线 |

**设计规范：**
- 单元大小：16×16px（单个T形转折）
- 排列：水平/垂直连续拼接
- 不透明度在 hover 态提升至 0.5
- 使用 `border-image` 或 SVG 背景实现

---

## 10.3 龙纹（Dragon Motif）

| 属性 | 值 |
|------|-----|
| 样式 | 五爪金龙侧影 |
| 颜色 | `rgba(212,168,67,0.06)` |
| 使用场景 | Hero 背景暗纹、玉玺表面浮雕、页面 Loading 动画 |
| 动效 | 金龙流光——轮廓线从龙头到龙尾逐段点亮（3s 循环） |

**设计规范：**
- 龙身：五段式（头→身→爪→尾→焰）
- 高亮流动方向：龙头至龙尾（3s 完成，无缝循环）
- Loading 态：100% 透明度，旋转 + 流光
- 背景态：6% 透明度，固定暗纹

---

## 10.4 汉砖纹（Han Brick Pattern）

| 属性 | 值 |
|------|-----|
| 样式 | 错缝排列矩形砖块 |
| 颜色 | 深黛底 `#1a1f2e` + 金线 `rgba(212,168,67,0.15)` |
| 使用场景 | Footer 背景、侧栏底部 |

**设计规范：**
- 砖块尺寸：120×40px
- 排列方式：错缝（每行偏移 60px）
- 分隔线：1px 金线
- 横向无缝平铺

---

## 10.5 海水江崖（Sea-Wave Cliff）

| 属性 | 值 |
|------|-----|
| 样式 | 层叠波浪 + 中央耸立山崖 |
| 颜色 | 蓝金渐变色（石青 `#3b82f6` → 鎏金 `#d4a843`） |
| 使用场景 | 侧栏底部装饰、Dashboard 页脚 |

**设计规范：**
- 波浪层数：3 层（由近及远逐层变淡）
- 山崖位置：水平居中，垂直高度 60px
- 渐变方向：底（石青）→ 顶（鎏金），从上到下
- 尺寸：100% × 120px（区域高度）
